# Hospital Chatbot (Cura AI)

A dedicated AI-powered chatbot system designed to enhance hospital patient services and streamline appointment inquiries.

## Project Structure
- **/api**: Backend logic and API endpoints.
- **/inc**: Configuration and authentication modules.
- **/public**: Frontend assets (HTML, CSS, JS).
- **/sql**: Database schemas and seed data.

## Technical Stack
- PHP, JavaScript, CSS, MySQL.

## Setup Instructions
1. Ensure XAMPP is running (Apache & MySQL).
2. Clone/download this repository to your `htdocs` directory.
3. Import the files from the `/sql` folder into your database (phpMyAdmin).
