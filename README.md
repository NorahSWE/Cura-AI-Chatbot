# Cura-AI-Chatbot
Cura AI Chatbot  A web-based AI chatbot designed to streamline patient inquiries and optimize hospital service delivery.

## Features
- Automated patient-provider communication.
- Web-based interface for easy access.
- Designed to reduce manual processing time for staff.

## Technical Stack
- Backend/Frontend: PHP
- Database: MySQL (managed via XAMPP)
- Integration: AI/LLM logic

## How to Run
1. Ensure [XAMPP](https://www.apachefriends.org/index.html) is installed on your machine.
2. Clone this repository into your htdocs folder.
3. Start Apache and MySQL via the XAMPP Control Panel.
4. Import the provided .sql database file into phpMyAdmin.
5. Access the project via http://localhost/Cura_AI.
